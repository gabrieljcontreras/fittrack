# FitTrack

A modern fitness tracking application designed to help you monitor and improve your workout performance.

## Overview

FitTrack is a full-stack web application that enables users to track their workouts, monitor progress, and maintain a consistent fitness routine. Built with a clean and intuitive interface, FitTrack makes fitness tracking simple and effective. 

## Features

- **Workout Logging** - Record your exercises, sets, reps, and weights
- **Progress Tracking** - Monitor your fitness journey over time
- **User Authentication** - Secure login and personalized experience
- **Responsive Design** - Access your workouts from any device
- **AI Trainer** - Connection to Claude API for personalized fitness advice to each user

## Tech Stack

### Frontend
- React
- JavaScript
- CSS

### Backend
- Node.js
- Express
- RESTful API

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/gabrieljcontreras/fittrack.git
cd fittrack
```

2. Install dependencies
```bash
npm install
```

3. Set up the client
```bash
cd client
npm install
```

4. Set up the server
```bash
cd server
npm install
```

### Running the Application

1. Start the backend server
```bash
cd server
npm start
```

2. Start the frontend client
```bash
cd client
npm start
```

The application will be available at `http://localhost:3000`
